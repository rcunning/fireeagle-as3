Class FireEagleConfigProperties | Constants
 
Packagenet.yahoo.fireeagle
Classpublic class FireEagleConfig

A class for the strings used to generate Fire Eagle API calls. You can override most of these strings if you want to point the library to different servers or uses different paths for method calls. Be sure to regenerate derrived API_PATH strings if you change API_PATH or any API_METHOD strings.
