Classpublic class XMPPConnection
InheritanceXMPPConnection flash.events.EventDispatcher
ImplementsIFireEagleRealtime

A class for connecting to an XMPP server to subscribe and listen for the Fire Eagle user location updates. A FireEagleEvent.USER_SUCCESS is dispatched for each subscribed user location update.
